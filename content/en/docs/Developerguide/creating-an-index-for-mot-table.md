# Creating an Index for MOT Table<a name="EN-US_TOPIC_0260578552"></a>

Standard PostgreSQL create and drop index statements are supported.

For example

```
create index  text_index1 on test(x) ;
The following is a complete example of creating an index for the ORDER table in a TPC-C  workload â€“
create FOREIGN table bmsql_oorder (
  o_w_id       integer      not null,
  o_d_id       integer      not null,
  o_id         integer      not null,
  o_c_id       integer not null,
  o_carrier_id integer,
  o_ol_cnt     integer,
  o_all_local  integer,
  o_entry_d    timestamp,
  primary key (o_w_id, o_d_id, o_id)
);
create index  bmsql_oorder_index1 on bmsql_oorder(o_w_id, o_d_id, o_c_id, o_id) ;
```
