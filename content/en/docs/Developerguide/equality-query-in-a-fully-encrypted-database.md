# Equality Query in a Fully-encrypted Database<a name="EN-US_TOPIC_0295141455"></a>

## enable\_full\_encryption<a name="section18307271684"></a>

**Parameter description**: Specifies whether the fully-encrypted database takes effect.

This parameter is a BACKEND parameter. Set it based on instructions provided in  [Table 1](en-us_topic_0289899927.md#en-us_topic_0283137176_en-us_topic_0237121562_en-us_topic_0059777490_t91a6f212010f4503b24d7943aed6d846).

**Value range**: Boolean

-   **on**: The fully-encrypted database is enabled.
-   **off**: The fully-encrypted database is disabled.

**Default value**:  **off**
