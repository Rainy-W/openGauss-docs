# Viewing Objects<a name="EN-US_TOPIC_0241704261"></a>

-   View help information.

    ```
    \?
    ```

-   Switch databases.

    ```
    \c dbname
    ```

-   List databases.

    ```
    \l
    ```


-   List tables.

    ```
    \dt
    ```


-   View a table structure.

    ```
    \d tablename
    ```


-   List schemas.

    ```
    \dn
    ```


-   View indexes.

    ```
    \di
    ```
