# kadmin.local<a name="EN-US_TOPIC_0249632275"></a>

## Function<a name="en-us_topic_0237152429_section10172172474817"></a>

**kadmin.local**, provided by the cluster authentication service Kerberos, is used to directly access the  **kdc**  service database and perform operations such as adding, deleting, and modifying Kerberos users.

## Parameter Description<a name="en-us_topic_0237152429_en-us_topic_0059778103_se0ebdb25733f42f5b4286a1fc7e54a13"></a>

The Kerberos tool is provided by an open-source third party. For details about the parameters, see the Kerberos official document at  [https://web.mit.edu/kerberos/krb5-1.17/doc/admin/admin\_commands/index.html](https://web.mit.edu/kerberos/krb5-1.17/doc/admin/admin_commands/index.html).
