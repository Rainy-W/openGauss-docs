# 获取帮助<a name="ZH-CN_TOPIC_0289900136"></a>

使用SQLdiag工具前，您可以通过以下指令获取帮助。

```
python main.py --help 
```

显示如下帮助信息：

```
usage: main.py [-h] -m {train,predict} -f FILE

sqldiag

optional arguments:
  -h, --help            show this help message and exit
  -m {train,predict}, --mode {train,predict}
  -f FILE, --file FILE
```
