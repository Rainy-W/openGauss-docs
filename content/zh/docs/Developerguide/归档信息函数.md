# 归档信息函数

归档信息函数表示可以用于获取归档状态的函数的函数。

- gs_get_archive_status()

  描述：获取归档状态列表。

  返回类型：text, text, timestamptz, text

  示例：

  ```
  postgres=# select * from gs_get_archive_status();
   archive_standby | archive_lsn | last_arch_time | archive_path 
  -----------------+-------------+----------------+--------------
  (0 rows)
  ```
