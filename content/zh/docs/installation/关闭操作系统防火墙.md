## 操作步骤<a name="zh-cn_topic_0241802566_zh-cn_topic_0085434636_zh-cn_topic_0059782018_section1388712913618"></a>

目前仅支持在防火墙关闭的状态下进行安装。

1.  检查防火墙是否关闭。

    ```
    systemctl status firewalld
    ```

    若防火墙未关闭，请执行[2](#zh-cn_topic_0241802566_zh-cn_topic_0085434636_zh-cn_topic_0059782018_li11887129193617)；

    若防火墙已关闭，则无需再关闭防火墙。

2.  <a name="zh-cn_topic_0241802566_zh-cn_topic_0085434636_zh-cn_topic_0059782018_li11887129193617"></a>关闭防火墙。

    ```
    systemctl stop firewalld.service
    ```

3.  修改/etc/selinux/config文件中的“SELINUX“值为“disabled“。
    1.  使用VIM打开config文件。

        ```
        vim /etc/selinux/config
        ```

    2.  修改“SELINUX“的值“disabled“。

        ```
        SELINUX=disabled
        ```

4.  重新启动操作系统。

    ```
    reboot
    ```

5.  在其他主机上重复步骤1到步骤3。
