# 如何清除编译过程中生成的临时文件<a name="ZH-CN_TOPIC_0241496977"></a>

## 问题<a name="section8138144317480"></a>

如何清除编译过程中生成的临时文件。

## 回答<a name="section09254965110"></a>

进入/sda/openGauss-server目录，执行如下命令清除编译过程中生成的临时文件。

-   删除由configure和make生成的文件。

    ```
    make distclean -sj
    ```

-   删除make生成的文件。

    ```
    make clean -sj
    ```
