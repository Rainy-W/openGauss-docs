# 产品特点<a name="ZH-CN_TOPIC_0241702871"></a>

openGauss相比其他开源数据库主要有复合应用场景，、高性能和高可用等产品特点。

-   复合应用场景
    -   行存储，支持业务数据频繁更新场景。
    -   列存储，支持业务数据追加和分析场景。
    -   内存表，支持高吞吐，低时延，极高性能场景。

-   高性能
    -   通过多核数据结构，增量检查点，大内存缓冲区管理实现百万级tpmC。
    -   服务端连接池，支持万级并发。

-   高可用
    -   支持主备同步，异步多种部署模式。
    -   数据页CRC校验，损坏数据页通过备机自动修复。
    -   备机并行恢复，10秒内可升主提供服务。
