# 查看对象<a name="ZH-CN_TOPIC_0241704261"></a>

-   查看帮助信息

    ```
    \?
    ```

-   切换数据库

    ```
    \c dbname
    ```

-   列举数据库

    ```
    \l
    ```


-   列举表

    ```
    \dt
    ```


-   查看表结构

    ```
    \d tablename
    ```


-   列举schema

    ```
    \dn
    ```


-   查看索引

    ```
    \di
    ```
