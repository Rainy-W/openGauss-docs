# 软件架构<a name="ZH-CN_TOPIC_0241702872"></a>

openGauss主要包含了openGauss服务器，客户端驱动，OM等模块，本章将介绍这些模块之间的相互关系。

openGauss软件的架构如[图1](#f078d2fc7ab574c8aa05c4a988e0df1b3)所示。在openGauss的文档中，将openGauss服务器称为实例。
